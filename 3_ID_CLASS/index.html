ID e CLASS

ID

O ID funciona como CPF. Assinm como só posso ter um número de CPF, cada elemento só pode ter um valor de ID. O CPF é único, ou seja, ninguém pode ter um CPF igual ao meu. Isso significa que o ID também é único, então não é possível repetir o mesmo valor duas vezes.

CLASS

O CLASS funciona de maneira oposta ao ID. DIferente do ID em que você só pode ter um valor, no CLASS você pode ter vários valores. Tmabém diferente do ID, em que não se pode repetir um valor, no CLASS vocÊ pode utilizar o mesmo valor para vários elemntos.
